## Computer Quiz Game (Python)

A simple command-line quiz game built with Python. This project was created while relearning Python fundamentals and was inspired by a tutorial from Tech with Tim. I customized parts of the logiv and added my own modifications to practice programming concepts.

## Features
- 🎀 Interactive quiz game in the terminal
- 🎀 Multiple computer-related questions
- 🎀 Score tracking system
- 🎀 Percentage calculation
- 🎀 Personalized feedback based on performance
- 🎀 Input validation before starting the game

## Built With
- 🎀 Python

2. Download or clone the project.
3. Open a terminal in the project folder.
4. Run: python quizGame.py
